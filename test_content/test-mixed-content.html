API Documentation

This page contains multiple JSON examples for our API endpoints.

User Data Endpoint

Returns user information in the following format:

{
  "user": {
    "id": 123,
    "name": "John Doe",
    "email": "john@example.com",
    "active": true,
    "roles": ["user", "admin"],
    "lastLogin": "2023-12-01T10:30:00Z"
  }
}

Product Catalog Endpoint

Returns product information:

{
  "products": [
    {
      "id": "prod-001",
      "name": "Laptop Computer",
      "price": 999.99,
      "category": "Electronics",
      "inStock": true,
      "specifications": {
        "processor": "Intel i7",
        "memory": "16GB",
        "storage": "512GB SSD"
      }
    },
    {
      "id": "prod-002",
      "name": "Wireless Mouse",
      "price": 29.99,
      "category": "Accessories",
      "inStock": false,
      "specifications": {
        "connectivity": "Bluetooth",
        "batteryLife": "6 months"
      }
    }
  ],
  "pagination": {
    "page": 1,
    "totalPages": 5,
    "totalItems": 47
  }
}

Error Response Format

When errors occur, the API returns:

{ "error": { "code": 404, "message": "Resource not found", "details": "The requested user ID does not exist", "timestamp": "2023-12-01T10:30:00Z" } }

For more information, please refer to our complete API documentation.
